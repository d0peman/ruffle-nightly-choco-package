# Ruffle Nightly Chocolatey Package (ruffle-nightly)

This repository contains the Chocolatey packaging files for **Ruffle Nightly**.

Ruffle is a Flash Player emulator written in Rust that allows Flash content to run natively without Adobe Flash.

Upstream project:
https://github.com/ruffle-rs/ruffle

This repository only contains the scripts and metadata used to build the Chocolatey package.

The package is automatically updated using GitHub Actions when new nightly builds are published.

Chocolatey package:
https://community.chocolatey.org/packages/ruffle-nightly
